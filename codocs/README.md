# Codocs 模块

**作者**： 周光营
**版本**： 1.0.0
**日期**： 2026-01-18

Codocs 模块是huizhi-yun 项目中的一个模块，为公司提供文档统一管理功能，为员工提供文档协同编辑服务。

文档集中存储在阿里云OSS中，分为用户私有文档、部门共享文档、项目文档、公司发布文档、知识库文档、产品文档等六类，其中前3类为工作文档，后3类为发布文档。工作文档为公司内部使用，发布文档为公司对内（公司发布文档和知识库文档）/对外（产品文档）发布使用。发布文档通过Nuxt Content框架实现。

文档协同编辑基于Milkdown实现，辅以AI辅助编辑。可实现多人在线协同编辑，支持实时预览、版本控制、历史记录、权限控制等功能。
